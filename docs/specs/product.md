# Product Spec

## 一、项目目标

实现一个员工管理前后端网站，用于管理员工基础信息。

这个项目也是为了练习 AI Agent 协作开发流程，包括：

- plan-agent 负责拆任务
- code-agent 负责写代码
- review-agent 负责审核代码

## 二、第一版功能

第一版包含：

1. 员工列表
2. 新增员工
3. 编辑员工
4. 删除员工
5. 根据姓名搜索员工
6. 根据部门筛选员工
7. 分页查询员工
8. 部门列表

## 三、第一版暂不做

第一版暂时不做：

1. 登录
2. 权限管理
3. 文件上传
4. Excel 导入导出
5. 操作日志
6. 多租户
7. 复杂审批流程

## 四、页面规划

第一版页面：

1. 员工管理页面
2. 员工新增弹窗
3. 员工编辑弹窗
4. 删除确认弹窗

## 五、用户操作流程

员工列表流程：

1. 用户打开员工管理页面
2. 前端请求员工分页接口
3. 后端返回员工列表
4. 前端展示表格

新增员工流程：

1. 用户点击新增按钮
2. 前端弹出新增表单
3. 用户填写员工信息
4. 前端提交新增接口
5. 后端校验并保存
6. 前端刷新员工列表

删除员工流程：

1. 用户点击删除按钮
2. 前端弹出确认框
3. 用户确认删除
4. 前端调用删除接口
5. 后端执行逻辑删除
6. 前端刷新员工列表
